Arrow Fucntion
===============
-compact syntac {reduces the number of lines of code (hardly 1 line of code)}
- implicit return (no need to wirte return sometimes yes)
-no fucntion keyword 
- no body {}
-doesn't have it own 'this' value,; it uses 'this' value from lexical/(outer) scope 
-'this' value is always static 


'this' in function
==========================
Regular : 'this' - how /where the func is called 
Arrow: 'this' - where the fucntion is written

============

-not be used if we expect this value to be dynamic inside a func
